User Profile Viewer

The backend profile endpoint trusts only the userId query parameter, with no session or ownership checks.

Load a Profile

In a secure app this page would show only your own account. Here, you can directly jump to any user by ID, including the admin profile.

Example exploits with Indian demo users:
/profile?userId=1 → Dixit (Owner / admin) profile with trophy
/profile?userId=2 → Rajesh’s customer profile
/profile?userId=3 → Priya’s customer profile
Any ID from 1 to 10 will load a different Indian user created in SQLite.
